# Yojana Mitra AI 3.0

AI-powered Welfare Scheme Recommendation System built using Flask, SQLite, HTML, CSS and JavaScript.

This project helps users find eligible government schemes based on:

- Age
- Income
- Category / Occupation

It also includes:

- Voice Assistant
- AI Chat Assistant
- Dashboard Analytics
- Admin Panel
- SQLite Database



# Features

## Home Page

Users enter:

- Age
- Annual Income
- Occupation

Then system recommends matching welfare schemes.



## Smart Scheme Matching

Based on:

- Age eligibility
- Income limit
- Category type

System gives:

- Recommended schemes
- Eligibility score percentage



## Voice Assistant

Users can speak:

I am a farmer age 45 income 2 lakh

System understands voice input and helps users quickly.

Supports multiple languages:

- English
- Hindi
- Kannada
- Tamil
- Telugu
- Marathi
- Bengali
- Gujarati
- Punjabi



## AI Chat Assistant

Users can ask:

- I am student
- I am farmer
- Senior citizen schemes

System replies with best suggestions.



## Dashboard

Displays:

- Total Users
- Matches Generated
- Success Rate
- Popular Categories
- System Insights



## Admin Panel

Admin can:

- Add new schemes
- View all schemes
- Manage data
